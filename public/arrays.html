Arrays

Arrays are fixed-size data structures that hold elements of the same type. Unlike lists, arrays in Kotlin have a fixed size once initialized, making them more memory-efficient in cases where the number of elements is known and doesn’t change. Kotlin supports both generic arrays and primitive arrays.

  1. Generic Array (Array<T>): Holds elements of any type (Int, String, Boolean, etc.).

  2. Primitive Arrays: Optimized for performance with specific types, such as:

    • IntArray

    • DoubleArray

    • BooleanArray

    • ByteArray

    • ...

Syntax

val intArray: Array<Int> = arrayOf(1, 2, 3, 4) // Type-safe array of Ints val stringArray: Array<String> = arrayOf("a", "b") // Type-safe array of Strings

Creating Arrays

Kotlin provides several ways to create arrays:

// Creating a generic array with values val genericArray = arrayOf("apple", "banana", "cherry") // Creating an empty array with a specific size val emptyArray = arrayOfNulls<String>(3) // Array of size 3 with all nulls // Creating a primitive array val intArray = intArrayOf(1, 2, 3, 4, 5) val doubleArray = doubleArrayOf(1.0, 2.0, 3.0)

Common Array Operations

Accessing Elements

Array elements can be accessed by their index, starting from 0.

val element = genericArray[1] // Accesses "banana"

Updating Elements

You can modify elements in an array by their index (for both generic and primitive arrays).

genericArray[1] = "blueberry" // Updates the second element to "blueberry"

Array Length

The size property returns the number of elements in the array.

val length = genericArray.size // 3

Iterating Over an Array

There are several ways to iterate over array elements.

  • For Loop

for (element in genericArray) { println(element) }

  • For Each

genericArray.forEach { println(it) }

  • Using Indices

for (index in genericArray.indices) { println("Element at index $index is ${genericArray[index]}") }

Array Transformation and Filtering

Kotlin offers several transformation and filtering functions that work on arrays, similar to those used with lists.

Mapping Over Elements

The map function creates a new list based on transformations applied to each array element.

val lengthArray = genericArray.map { it.length } // [5, 10, 6] (for "apple", "blueberry", "cherry")

Filtering Elements

The filter function creates a new list with elements that satisfy a given condition.

val filteredArray = intArray.filter { it % 2 == 0 } // [2, 4] (for elements that are even)

Transforming with mapIndexed

The mapIndexed function provides both index and element in transformations.

val indexLengthArray = genericArray.mapIndexed { index, value -> "$index: ${value.length}" }

Array Manipulation Functions

Kotlin has various array manipulation functions for sorting, reversing, and other operations.

Sorting

You can sort arrays in ascending or descending order.

val sortedArray = intArray.sortedArray() // Returns a new sorted IntArray val descendingArray = intArray.sortedArrayDescending()

Reversing

The reversedArray function returns a new array with elements in reverse order

val reversedArray = genericArray.reversedArray()

Finding Elements

Kotlin provides several functions for finding elements.

  • First and Last Elements

val firstElement = genericArray.first() // "apple" val lastElement = genericArray.last() // "cherry"

  • Finding Elements by Condition

val firstEven = intArray.firstOrNull { it % 2 == 0 } // Finds the first even number or null if none

Checking Conditions

Kotlin includes functions for checking array-wide conditions.

val allEven = intArray.all { it % 2 == 0 } // Returns true if all elements are even val anyEven = intArray.any { it % 2 == 0 } // Returns true if any element is even val containsFive = intArray.contains(5) // Checks if the array contains the number 5

Copying and Converting Arrays

Copying Arrays

The copyOf function creates a new array with the same elements.

val copiedArray = intArray.copyOf()

Converting Arrays to Lists

The toList function converts an array to a list.

val arrayList = intArray.toList()

Joining Elements into a String

The joinToString function joins elements into a single string with a specified separator.

val joinedString = intArray.joinToString(", ") // "1, 2, 3, 4, 5"

Primitive Array Types

Kotlin’s primitive array types (IntArray, DoubleArray, etc.) are optimized for performance and avoid the overhead of boxing (wrapping in objects). They support most of the same functions as generic arrays.

val doubleArray = doubleArrayOf(1.5, 2.5, 3.5) val intArray = IntArray(5) { it * 2 } // Creates an IntArray with [0, 2, 4, 6, 8]
