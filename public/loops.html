Loops

Kotlin provides several types of loops for repetitive tasks: for, while, and do-while. Each serves different needs, allowing you to iterate over collections, ranges, or perform repeated actions under specific conditions.

For Loop

The for loop in Kotlin is used to iterate over a range, an array, a list, or any other iterable object

for (item in collection) { // Code to execute for each item }

Example: Iterating Over a Range

for (i in 1..5) { println(i) // Output: 1 2 3 4 5 }

Example: Iterating Over a Range

val fruits = listOf("Apple", "Banana", "Cherry") for (fruit in fruits) { println(fruit) } // Output: Apple, Banana, Cherry

Additional Control in for Loops

  • step: Specify increments for the range.

  • downTo: Loop in reverse order.

  • until: Exclude the upper bound in the range.

for (i in 10 downTo 1 step 2) { println(i) // Output: 10 8 6 4 2 }

Iterating with Indices

val fruits = listOf("Apple", "Banana", "Cherry") for (index in fruits.indices) { println("Index $index: ${fruits[index]}") } // Output: Index 0: Apple, Index 1: Banana, Index 2: Cherry

withIndex Extension Function

for ((index, value) in fruits.withIndex()) { println("Index $index: $value") } // Output: Index 0: Apple, Index 1: Banana, Index 2: Cherry

While Loop

The while loop executes as long as a given condition is true. It’s ideal when you don’t know in advance how many times the loop should run.

while (condition) { // Code to execute while condition is true }

Example

var i = 1 while (i <= 5) { println(i) i++ } // Output: 1 2 3 4 5

Infinite while Loop

An infinite loop can be created if the condition always evaluates to true. Be cautious as this can lead to application crashes or high CPU usage:

while (true) { println("This will print indefinitely unless stopped.") }

Do not use infinite loops, It can lead to memory leaks, applications crashes or high memory and CPU usage.

do-while Loop

The do-while loop is similar to the while loop but guarantees at least one execution, as it checks the condition after the loop body.

do { // Code to execute } while (condition)

Example

var i = 1 do { println(i) i++ } while (i <= 5) // Output: 1 2 3 4 5

Control Statements in Loops

Kotlin offers several statements to control loop execution:

break

Exits the loop immediately.

for (i in 1..5) { if (i == 3) break println(i) } // Output: 1 2

continue

Skips the current iteration and moves to the next.

for (i in 1..5) { if (i == 3) continue println(i) } // Output: 1 2 4 5

Labels in Loops

Labels help manage nested loops, allowing you to specify which loop to break or continue.

outer@ for (i in 1..3) { for (j in 1..3) { if (i == 2 && j == 2) break@outer println("i: $i, j: $j") } } // Output: i: 1, j: 1, i: 1, j: 2, i: 1, j: 3, i: 2, j: 1
