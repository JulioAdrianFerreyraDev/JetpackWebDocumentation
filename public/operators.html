Operators

Kotlin Operators Documentation Table

Category

Operator

Description

Example

Arithmetic Operators

+

Adds two values.

val sum = a + b

-

Subtracts the second value from the first.

val diff = a - b

*

Multiplies two values.

val product = a * b

/

Divides the first value by the second.

val quotient = a / b

%

Returns the remainder of a division.

val remainder = a % b

Comparison Operators

==

Checks if two values are equal.

if (a == b) { ... }

!=

Checks if two values are not equal.

if (a != b) { ... }

>

Checks if the first value is greater than the second.

if (a > b) { ... }

<

Checks if the first value is less than the second.

if (a < b) { ... }

>=

Checks if the first value is greater than or equal to the second.

if (a >= b) { ... }

<=

Checks if the first value is less than or equal to the second.

if (a <= b) { ... }

Assignment Operators

=

Assigns a value to a variable.

val x = 5

+=

Adds the right operand to the left operand and assigns the result to the left operand.

x += 2

-=

Subtracts the right operand from the left operand and assigns the result.

x -= 2

*=

Multiplies and assigns the result.

x *= 2

/=

Divides and assigns the result.

x /= 2

%=

Finds the remainder and assigns the result.

x %= 2

Logical Operators

&&

Returns true if both conditions are true (AND).

if (a > 5 && b > 5) { ... }

| |

Returns true if at least one condition is true (OR).

if (a > 5 || b > 5) { ... }

!

Reverses the boolean value (NOT).

if (!isAvailable) { ... }

Unary Operators

+

Indicates a positive value.

val pos = +a

-

Indicates a negative value.

val neg = -a

++

Increments the value by 1 (prefix/postfix).

a++ or ++a

--

Decrements the value by 1 (prefix/postfix).

a-- or --a

Bitwise Operators

shl

Left shift operator.

val result = a shl 1

shr

Right shift operator.

val result = a shr 1

and

Performs a bitwise AND operation.

val result = a and b

or

Performs a bitwise OR operation.

val result = a or b

xor

Performs a bitwise XOR operation.

val result = a xor b

inv

Inverts all bits.

val result = a.inv()

Range Operators

..

Creates a range from the first value to the second.

for (i in 1..5) { ... }

step

Specifies a step for iteration in a range.

for (i in 1..10 step 2) { ... }

downTo

Creates a range that decrements.

for (i in 5 downTo 1) { ... }

Equality Operators

===

Checks if two references point to the same object.

if (a === b) { ... }

!==

Checks if two references do not point to the same object.

if (a !== b) { ... }

Notes

  • Operators in Kotlin can be overloaded for custom types.

  • Logical operators and comparison operators return boolean values.

  • Use ranges (..) for looping in a defined range.
