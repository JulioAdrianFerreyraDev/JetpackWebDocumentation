Android Studio Recommendations

Shortcuts

Using the Android no Keymap Proper Redo keymap:

Code

  • Line duplication: Ctrl + K

  • Delete Line: Ctrl + Alt + Ñ

  • Add caret: Ctrl + Alt + Key up/Key Down

  • Move line: Mayus + Alt + Key Up/Key Down

  • Tab navigation: Alt + Key Left/Key Right

  • Add new Directory Folder: Ctrl + ¿

  • New Kotlin class / File Ctrl + Ñ

  • Run application: Mayus + F10

  • Document Formating: Ctrl + Alt + L

Editor

Close tabs:

  • All: Ctrl + Alt + F2

  • All except the one in use: Ctrl + Alt + F3

  • The one in use: Ctrl + F4

  • Change cursor focus: Ctrl + Tab

Files

  • Search File: Ctrl + Mayus + N

  • Rename (Refactor): Mayus + F6

Search and Replace:

  • Search: Ctrl + F

  • Replace: Ctrl + R
