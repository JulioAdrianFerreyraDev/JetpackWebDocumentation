if-else statement

In Kotlin, if is an expression, which means it returns a value. This sets Kotlin apart from many other programming languages where if is a statement but not an expression. With if expressions, you can directly assign the result to variables, making your code cleaner and more concise.

Syntax of if Statement

The syntax of a basic if statement in Kotlin is as follows:

if (condition) { // Code to execute if condition is true }

You can also add an else block to handle cases when the condition is false:

if (condition) { // Code to execute if condition is true } else { // Code to execute if condition is false }

For multiple conditions, else if can be used:

if (condition1) { // Code to execute if condition1 is true } else if (condition2) { // Code to execute if condition2 is true } else { // Code to execute if all conditions are false }

Using if as an Expression

Since if in Kotlin is an expression, you can assign its result to a variable directly:

val max = if (a > b) a else b

In this example, max is assigned the value of a if a is greater than b; otherwise, it is assigned the value of b.

Nesting if Statements

You can nest if statements inside each other for more complex conditions:

val age = 20 val hasID = true if (age >= 18) { if (hasID) { println("Access granted") } else { println("ID required") } } else { println("Access denied") }

Single-line if expression

If the result of the if/else statement block is a single line of code, you can skip the {} brackets

if (condition) println("Hello World")
