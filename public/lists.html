Lists

In Kotlin, a list is an ordered collection of elements. There are two main types of lists:

  1. Immutable List (List<T>): Read-only, meaning elements cannot be added or removed.

  2. Mutable List (MutableList<T>): Supports modification (adding, removing, updating elements).

Creating Lists

Kotlin offers several ways to create lists:

// Immutable List val immutableList = listOf(1, 2, 3) // Mutable List val mutableList = mutableListOf(1, 2, 3)

Syntax

val intList: List<Int> = listOf(1, 2, 3) // Immutable type-safe list of Ints val mutableIntList: MutableList<Int> = mutableListOf() // Mutable type-safe list of Ints mutableIntList.add(5) // Works, as 5 is an Int // Attempting to add a String to intList or mutableIntList would cause a compilation error.

Common List Operations

Accessing Elements

Elements in a list can be accessed by their index:

val firstElement = immutableList[0] // Access the first element val elementAt = immutableList.getOrNull(2) // Safely get an element or return null

Adding and Removing Elements (Mutable Lists Only)

For mutable lists, you can add and remove elements as needed.

// Adding elements mutableList.add(4) // Adds to the end mutableList.add(1, 5) // Adds 5 at index 1 // Removing elements mutableList.remove(2) // Removes the first occurrence of 2 mutableList.removeAt(0) // Removes the element at index 0 mutableList.clear() // Removes all elements

Updating Elements (Mutable Lists Only)

mutableList[1] = 10 // Sets the value at index 1 to 10

Iterating Over a List

Kotlin provides multiple ways to iterate over a list.

  • For Loop

for (element in immutableList) { println(element) }

  • For Loop with Indices

for (index in immutableList.indices) { println("Index $index: ${immutableList[index]}") }

  • Using forEach

immutableList.forEach { println(it) }

  • Reverse Iteration

for (index in immutableList.indices.reversed()) { println(immutableList[index]) }

List Transformation Functions

Kotlin has various functions for transforming lists, such as map, filter, and sorted.

  • Map: Transforms each element using a given function.

val doubled = immutableList.map { it * 2 } // [2, 4, 6]

  • Filter: Returns a list with elements that match a condition.

val evenNumbers = immutableList.filter { it % 2 == 0 } // [2]

  • Sorted Returns a sorted list.

val sortedList = immutableList.sorted() // [1, 2, 3]

Note that sorted produces a new list, while sort modifies the original list

Checking Conditions

Kotlin provides functions to check conditions in a list.

  • Contains: Checks if a list contains a specific element.

val containsTwo = immutableList.contains(2) // true

* **All**, **Any**, **None**: Checks conditions for elements in a list.

val allPositive = immutableList.all { it > 0 } // true val anyEven = immutableList.any { it % 2 == 0 } // true val noneNegative = immutableList.none { it < 0 } // true

Finding Elements

You can find elements in a list by using functions like find, first, last, and indexOf.

  • Find: Returns the first matching element or null.

val firstEven = immutableList.find { it % 2 == 0 } // 2

  • First/Last: Returns the first or last element that matches a condition.

val firstElement = immutableList.first() // 1 val lastElement = immutableList.last() // 3

  • IndexOf/LastIndexOf: Finds the index of an element.

val indexOfTwo = immutableList.indexOf(2) // 1

Combining Lists

You can combine lists using + and - operators, or using functions like union, intersect, and subtract.

val combined = immutableList + listOf(4, 5) // [1, 2, 3, 4, 5] val subtracted = combined - listOf(1, 2) // [3, 4, 5]

Getting the List Size

Use the size property to get the length of the list:

val listSize = immutableList.size // 3

Copying Lists

To create a copy of a list, use toList (for an immutable copy) or toMutableList (for a mutable copy).

val copiedList = immutableList.toList() val mutableCopiedList = immutableList.toMutableList()
