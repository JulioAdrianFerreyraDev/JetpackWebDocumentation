Sets

Sets are unordered collections that do not allow duplicate elements. Kotlin provides two main types of sets:

  • Set (Set<T>): Immutable, read-only set.

  • MutableSet (MutableSet<T>): Allows adding and removing elements, and is both mutable and modifiable.

  • Both types of sets are generic and can hold any type (String, Int, custom objects, etc.).

Creating Sets

You can create sets using various methods, depending on whether the set is mutable or immutable

// Immutable Set val fruitSet = setOf("apple", "banana", "cherry") // Mutable Set val mutableFruitSet = mutableSetOf("apple", "banana", "cherry") // Empty Set val emptySet = emptySet<String>()

Syntax

val stringSet: Set<String> = setOf("apple", "banana") // Immutable type-safe set of Strings val mutableStringSet: MutableSet<String> = mutableSetOf() // Mutable type-safe set of Strings mutableStringSet.add("cherry") // Adds "cherry" to the set

Common Set Operations

Kotlin provides a range of operations for working with sets.

Accessing Elements

Elements in a set can’t be accessed by index, as sets are unordered. However, you can check for the presence of an element using the contains function.

val containsBanana = fruitSet.contains("banana") // true

Adding and Removing Elements (Mutable Sets Only)

Mutable sets support adding and removing elements

// Adding elements mutableFruitSet.add("orange") // Removing elements mutableFruitSet.remove("apple")

Set Size

The size property returns the number of unique elements in the set.

val setSize = fruitSet.size // 3

Iterating Over a Set

You can iterate over a set using a loop, forEach, or forEachIndexed if needed

for (fruit in fruitSet) { println(fruit) } // Using forEach fruitSet.forEach { println(it) }

Set Transformation and Filtering

Sets in Kotlin support a variety of transformation and filtering functions, similar to lists.

Mapping Elements

The map function creates a new list by transforming each element in the set.

val lengthList = fruitSet.map { it.length } // [5, 6, 6] for "apple", "banana", "cherry"

Filtering Elements

The filter function creates a new list with elements that satisfy a given condition.

val filteredSet = fruitSet.filter { it.length > 5 } // ["banana", "cherry"]

Checking for Empty

You can check if a set is empty with isEmpty() or isNotEmpty().

val isEmpty = fruitSet.isEmpty() // false val isNotEmpty = fruitSet.isNotEmpty() // true

Element Checking with all, any, and contains

Kotlin offers several functions for checking elements in a set.

  • All: Checks if all elements match a given condition.

  • Any: Checks if any element matches a given condition.

  • Contains: Checks if a specific element is in the set.

val allHaveFiveChars = fruitSet.all { it.length == 5 } // false val anyHasFiveChars = fruitSet.any { it.length == 5 } // true val containsApple = fruitSet.contains("apple") // true

Set Operations

Sets support a range of mathematical operations, such as union, intersection, and difference.

Union

The union function returns a new set containing all unique elements from both sets.

val set1 = setOf("apple", "banana") val set2 = setOf("banana", "cherry") val unionSet = set1 union set2 // ["apple", "banana", "cherry"]

Intersection

The intersect function returns a new set with elements common to both sets

val intersectionSet = set1 intersect set2 // ["banana"]

Difference

The subtract function returns a new set with elements from the first set that are not in the second set.

val differenceSet = set1 subtract set2 // ["apple"]

Symmetric Difference

Kotlin doesn't have a built-in symmetric difference function, but you can use union and subtract to find elements that exist in one set but not both.

val symmetricDifference = (set1 union set2) - (set1 intersect set2) // ["apple", "cherry"]

Joining Elements into a String

The joinToString function joins all elements into a single string with a specified separator.

val joinedString = fruitSet.joinToString(", ") // "apple, banana, cherry"

Converting Sets to Other Collections

You can convert sets to other types of collections, such as lists or arrays.

val listFromSet = fruitSet.toList() val arrayFromSet = fruitSet.toTypedArray()
