Firebase

Is a set of backend cloud computing services and application development platforms provided by Google. It hosts databases, services, authentication, and integration for a variety of applications, including Android, iOS, JavaScript, Node.js, Java, Unity, PHP, and C++.

Bill of Materials (BoM)

When you use the Firebase BoM (Bill of Materials) in your app, the BoM automatically pulls in the individual library versions mapped to BoM's version. All the individual library versions will be compatible. When you update the BoM's version in your app, all the Firebase libraries that you use in your app will update to the versions mapped to that BoM version.

Implementation

First, it is necessary to create an account in Firebase and access to the Firebase Console. Create a new Project and follow the steps given by Firebase:

  • Download the google-services.json and save it in te app directory of your project.

  • There are some implementations that can be done within the build.gradle files for both Project and Module

Project

plugins { ... id 'com.google.gms.google-services' version '4.4.2' apply false }
plugins { ... id("com.google.gms.google-services") version "4.4.2" apply false }

App Module

plugins { ... id 'com.google.gms.google-services' } ... dependencies { // Import the Firebase BoM implementation platform('com.google.firebase:firebase-bom:33.1.2') // More Firebase dependencies ... }
plugins { ... id("com.google.gms.google-services") } ... dependencies{ // Import the Firebase BoM implementation(platform("com.google.firebase:firebase-bom:33.1.2")) // More Firebase dependencies ... }

Note

When using the BoM, don't specify versions in Firebase dependencies.
