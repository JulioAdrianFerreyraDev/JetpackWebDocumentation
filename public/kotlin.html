Variables

There are two types of variables in Kotlin: Immutable and Mutable.

Immutables

Immutables are variables that are not truly variables but constants. This means that once they have a value, it cannot be changed.

They must be declared with the val keyword

val myName : String = "Kotlin" myName = "Android" // Throws an error

Mutables

These variables allow the value to change at runtime and must be declared using the var keyword.

var name : String = "Kotlin" name = "Android" // Valid

In essence, user val over var

Initializing Variables

There are two ways to initialize a variable:

  1. Using Strong-Typing:

val value : String

  1. Adding an initial value:

val value = "value"
