Intents

Intents are asynchronous messages that allow application components to request functionality from other Android components. They are used to start activities, services, and send broadcasts.

  • Starting a new activity: Use explicit intents to navigate between activities in your app.

  • Opening a web page: Use implicit intents to open URLs in a web browser.

  • Sending an email: Use implicit intents to open email clients with pre-filled content.

  • Broadcasting messages: Use broadcast intents to send messages to multiple app components.

Types

Types of Intents

Explicit Intents

Explicit intents specify the component to start by name (the fully qualified class name). They are typically used to start a component within the same application.

val intent = Intent(this, TargetActivity::class.java) startActivity(intent)

Implicit Intents

Implicit intents do not specify a specific component. Instead, they declare a general action to perform, allowing any app that can handle the action to respond.

val intent = Intent(Intent.ACTION_VIEW).apply { data = Uri.parse("https://www.example.com") } startActivity(intent)

Creating an intent involves creating an instance of the Intent class and specifying the action, data, and component (if applicable).

val intent = Intent(this, TargetActivity::class.java) intent.putExtra("key", "value") startActivity(intent)

Types of Intent Actions

Intent actions are predefined string constants that describe the general action to be performed by the intent. These actions determine how the intent is handled by the receiving component.

Intent.ACTION_VIEW

Displays data to the user.

Common Uses: Opening a web page in a browser, viewing a map location, viewing a contact, or opening a document.

Open a Web Page:

// If it necessary to launch the Intent where there is no context context = LocalContext.current val intent = Intent(Intent.ACTION_VIEW).apply { data = Uri.parse("https://www.example.com") } /*context.*/startActivity(intent)

**View a Location on a Map:**

// If it necessary to launch the Intent where there is no context context = LocalContext.current val gmmIntentUri = Uri.parse("geo:37.7749,-122.4194") val mapIntent = Intent(Intent.ACTION_VIEW, gmmIntentUri) mapIntent.setPackage("com.google.android.apps.maps") /*context.*/startActivity(mapIntent)

Intent.ACTION_SEND

Send data to another application.

Common Uses: Sharing text, images, or other content via email, messaging apps, or social media.

Share Text Content:

// If it necessary to launch the Intent where there is no context context = LocalContext.current val intent = Intent(Intent.ACTION_SEND).apply { type = "text/plain" putExtra(Intent.EXTRA_TEXT, "Check out this link: https://www.example.com") } /*context.*/startActivity(Intent.createChooser(intent, "Share via"))

Share an Image:

// If it necessary to launch the Intent where there is no context context = LocalContext.current val uri = Uri.parse("file://path/to/image") val intent = Intent(Intent.ACTION_SEND).apply { type = "image/ /* [.png .webp .jpg .jpeg * -> (All) ]*/" putExtra(Intent.EXTRA_STREAM, uri) } /*context.*/startActivity(Intent.createChooser(intent, "Share Image via"))

Intent.ACTION_SENDTO

Send data to a specific URI.

Common Uses: Sending an email or SMS.

Send an Email:

// If it necessary to launch the Intent where there is no context context = LocalContext.current val intent = Intent(Intent.ACTION_SENDTO).apply { data = Uri.parse("mailto:example@example.com") putExtra(Intent.EXTRA_SUBJECT, "Subject Here") putExtra(Intent.EXTRA_TEXT, "Body Here") } /*context.*/startActivity(Intent.createChooser(intent, "Email Client")) // startActivity(intent) by default
// If it necessary to launch the Intent where there is no context context = LocalContext.current // Send an email val intent = Intent(Intent.ACTION_SEND) email = "example@mail.com, test@example.com" var emailAddress: Array<String> = emptyArray() email.trim().split(", ").forEach { emailAddress += it } intent.putExtra(Intent.EXTRA_EMAIL, emailAddress) intent.putExtra(Intent.EXTRA_SUBJECT, "subject") intent.putExtra(Intent.EXTRA_TEXT, "message") intent.type = "message/rfc822" /*context.*/startActivity(Intent.createChooser(intent, "Email Client"))

Send an SMS:

// If it necessary to launch the Intent where there is no context context = LocalContext.current val intent = Intent(Intent.ACTION_SENDTO).apply { data = Uri.parse("smsto:$phoneNumber") putExtra("sms_body", "Hello, how are you?") } /*context.*/startActivity(intent)

Intent.ACTION_PICK

Pick an item from the data, returning what was selected.

Common Uses: Picking a contact, picking an image from the gallery.

Pick a Contact:

// If it necessary to launch the Intent where there is no context context = LocalContext.current val intent = Intent(Intent.ACTION_PICK).apply { type = ContactsContract.CommonDataKinds.Phone.CONTENT_TYPE } /*context.*/startActivityForResult(intent, REQUEST_CODE_PICK_CONTACT)

Pick an Image:

// If it necessary to launch the Intent where there is no context context = LocalContext.current val intent = Intent(Intent.ACTION_PICK).apply { type = "image/ * " // No blank spaces. } /*context.*/startActivityForResult(intent, REQUEST_CODE_PICK_IMAGE)

Intent.ACTION_DIAL

Dial a number, showing the dialer screen with the number pre-filled.

Common Uses: Initiating a phone call without directly placing the call.

Dial a Number:

// If it necessary to launch the Intent where there is no context context = LocalContext.current val intent = Intent(Intent.ACTION_DIAL).apply { data = Uri.parse("tel:$phoneNumber") } /*context.*/startActivity(intent)

Intent.ACTION_CALL

Directly call a number (requires CALL_PHONE permission).

Common Uses: Placing a phone call directly.

Directly Call a Number:

// If it necessary to launch the Intent where there is no context context = LocalContext.current val intent = Intent(Intent.ACTION_CALL).apply { data = Uri.parse("tel:$phoneNumber") } /*context.*/startActivity(intent)

Note

Using ACTION_CALL requires the android.permission.CALL_PHONE permission in your manifest and is generally discouraged in favor of ACTION_DIAL for security reasons.

Edit

Edit the specified data.

Common Uses: Editing a contact, editing a calendar event.

Edit a Contact:

// If it necessary to launch the Intent where there is no context context = LocalContext.current val contactUri = Uri.parse("content://contacts/people/1") val intent = Intent(Intent.ACTION_EDIT).apply { data = contactUri } /*context.*/startActivity(intent)
