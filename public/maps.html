Maps

Maps are collections of key-value pairs. Each key is unique, while values can be duplicated. In Kotlin, there are two main types of maps:

  • Mutable Map (MutableMap<K, V>): Allows modification (adding, removing, updating entries).

  • Immutable Map (Map<K, V>): A read-only map.

Creating Maps

Kotlin provides several ways to create maps:

// Immutable Map val immutableMap = mapOf("one" to 1, "two" to 2) // Mutable Map val mutableMap = mutableMapOf("one" to 1, "two" to 2)

Syntax

val stringIntMap: Map<String, Int> = mapOf("a" to 1, "b" to 2) // Immutable type-safe map val mutableStringIntMap: MutableMap<String, Int> = mutableMapOf() // Mutable type-safe map mutableStringIntMap["c"] = 3 // Adds key "c" with value 3

Common Map Operations

Accessing Values

Values in a map can be accessed by their keys.

val value = immutableMap["one"] // Returns 1 val safeValue = immutableMap["three"] // Returns null if key doesnâ€™t exist val valueOrDefault = immutableMap.getOrDefault("three", 0) // Returns 0 if key doesn't exist

Adding and Updating Entries (Mutable Maps Only)

For mutable maps, you can add new entries or update existing ones.

// Adding entries mutableMap["three"] = 3 mutableMap.put("four", 4) // Updating entries mutableMap["one"] = 10

Removing Entries (Mutable Maps Only)

You can remove entries from mutable maps by their key:

mutableMap.remove("two") // Removes the entry with key "two" mutableMap.clear() // Removes all entries

Checking for Keys and Values

Kotlin provides several functions to check for keys or values in a map.

// Check if a key exists val hasKey = immutableMap.containsKey("one") // true // Check if a value exists val hasValue = immutableMap.containsValue(1) // true

Iterating Over a Map

Kotlin provides several ways to iterate over a map’s entries, keys, and values.

  • For Loop with Entries

for ((key, value) in immutableMap) { println("$key = $value") }

  • For Each Entry

immutableMap.forEach { (key, value) -> println("$key = $value") }

  • Iterating Over Keys and Values Separately

for (key in immutableMap.keys) { println("Key: $key") } for (value in immutableMap.values) { println("Value: $value") }

Map Transformations

Kotlin offers transformation functions, such as map, filter, and mapValues.

  • Mapping Over Entries: Transforms each entry.

val newMap = immutableMap.map { (key, value) -> "$key -> ${value * 2}" } // Returns a List of transformed entries: ["one -> 2", "two -> 4"]

  • Mapping Only Values: Transform values, keeping the same keys.

val doubledValuesMap = immutableMap.mapValues { it.value * 2 } // {one=2, two=4}

  • Filtering Entries: Returns a new map with entries that match a condition.

val filteredMap = immutableMap.filter { (key, value) -> value > 1 } // {two=2}

Combining Maps

Maps can be combined using + or with specific functions like putAll (for mutable maps only).

  • Immutable Combination

val combinedMap = immutableMap + mapOf("three" to 3) // {one=1, two=2, three=3}

  • Mutable Combination

mutableMap.putAll(mapOf("five" to 5, "six" to 6)) // Modifies mutableMap in place

Getting Map Size

Use the size property to get the number of entries in a map.

val mapSize = immutableMap.size // 2

Default Values

The getOrElse function provides a default value if a key is not found.

val value = immutableMap.getOrElse("unknown") { 0 } // Returns 0 if "unknown" isn't found

Copying Maps

Use toMap or toMutableMap to create copies of maps.

val copiedMap = immutableMap.toMap() // Immutable copy val mutableCopiedMap = immutableMap.toMutableMap() // Mutable copy
