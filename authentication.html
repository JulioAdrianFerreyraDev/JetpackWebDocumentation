Firebase Authentication is a service provided by Firebase that enables you to authenticate users in your app. It supports multiple authentication methods, including email and password, phone number, and popular identity providers like Google, Facebook, Twitter, and GitHub.

Key Features

  1. Multiple Authentication Methods:

  • Email and Password: Allow users to sign up and log in using their email addresses and passwords.

  • Phone Number: Authenticate users by sending a verification code to their phone number.

  • Federated Identity Providers: Use third-party providers like Google, Facebook, Twitter, and GitHub to sign in users.

  1. Anonymous Authentication:

    • Allow users to access your app without requiring them to sign up. They can later upgrade to a full account by providing additional credentials.

  2. Secure Token Management:

    • Firebase Auth manages user sessions securely, providing access tokens that can be used to authenticate requests to your backend services.

  3. Easy Integration with Firebase Services:

    • Seamlessly integrate with other Firebase services like Firestore, Realtime Database, and Firebase Storage to manage user-specific data.

  4. Custom Authentication System:

    • Implement custom authentication by integrating with your existing user management system and issuing Firebase Auth tokens.

Implementation

In your module (app-level) Gradle file, add the dependency for the Firebase Authentication library for Android. Is recommend to use the Firebase Android BoM to control library versioning.

dependencies { // Import the BoM for the Firebase platform implementation(platform("com.google.firebase:firebase-bom:33.1.2")) // When using the BoM, you don't specify versions in Firebase library dependencies implementation("com.google.firebase:firebase-auth-ktx") }

Initialize Auth

  1. Declare an instance of FirebaseAuth

    private val auth: FirebaseAuth = Firebase.auth

  2. You can check if there is an active session

    var currentUser : FirebaseUser? = null private set private fun getCurrentUser(){ currentUser = auth.currentUser } init{ getCurrentUser() }

val user = auth.currentUser user?.let { // Name, email address, and profile photo Url val name = it.displayName val email = it.email val photoUrl = it.photoUrl // Check if user's email is verified val emailVerified = it.isEmailVerified // The user's ID, unique to the Firebase project. val uid = it.uid }
